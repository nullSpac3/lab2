Description of when to utilize the following HTML elements:

<tr>
used to define a row within a table

<th>
used to define a header row; where the text is highlighted in bold

<td>
used to define column elements

<thead>
used to group header content in a table; can be used to enable independent scrolling of table content

<tfoot>
used to group footer content in a table; can be used to enable independent scrolling of table content

<caption>
used to title a table

Colspan merges columns while rowspan merges rows


Cell Padding defines the space between a cell and its respective border
Cell Spacing defines the space between individual cells

To define table borders using CSS, create the following class:

table {
border: 1px solid;
}

To control border widths using CSS, create add the following poperty to the previous class:

table {
border: 1px solid;
border-width: 3px
}
